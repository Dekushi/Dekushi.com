Dekushi 卓球クラブでは、練習生を随時募集しています。

全日本選手権を初め、数々の全国大会出場経験者のコーチがあなたのレベルアップに、多大に貢献します。ジュニアでは県大会個人優勝者を初め、地区大会出場レベルの選手が、半年後は、関東大会の県代表決定戦迄のレベルになっております。社会人の方も、オープン大会等でも上位の成績を残せる実力が繰り返しの基礎練習で身に付けられます。

まだまだ、Dekushi卓球クラブの魅力を教えたいのですが、この先は是非ご自身で体験して頂けたらと思います🙇
